Intranet

This page lists things mostly of interest to Chicas staff and students. Note that this page is public, but not advertised. The services detailed won't be public.

Chicas Services

Most of these services will require a Chicas login. To request a Chicas login for yourself send an email to Barry with your email and ISS username. If you have a login already and want one for someone else then submit an issue on the Chicas IT helpdesk.

  • The Chicas Code gitlab development server.
  • The Chicas IT Helpdesk - add issues there.

Chicas Email

The Chicas mailing list is chicas@lists.lancs.ac.uk. Posting is restricted to list members, which is all staff and postgraduate students.

Do not send attachments on this list, the maximum message size is very small, and the mail will silently fail.

To see the members of the list, send an email to majordomo@lists.lancs.ac.uk with who chicas in the body of the message.

To change your email address on the list or to request new people, send names and email addresses to Barry.

Chicas Web Pages

Changes for the web server should be requested via the web source issue tracker, for which you need a github account, or the Chicas IT helpdesk, or via email to Barry.

Experts only If you have skills with HTML then you can fork the web source repository, edit and send a pull request. To build the site from source to test your changes requires the Hyde static site generator system installed on your computer.

A newer, easier, system for changes is under consideration.

Chicas News

If you have some news for the website, please send it to Barry with the copy text, any images, links etc. Please try and edit into a news-style report if possible.

Chicas Calendar

The Chicas calendar events which appear on the web site are managed by a shared Google Calendar. You can ask for permission to add events to the calendar - file a Chicas IT Helpdesk issue.

Chicas Graphics and Style

The Chicas logo and other graphics are kept in a Github repository. There are also some templates for LaTeX, Libre Office, and MS Office. If you want to contribute templates please contact Barry.

Chicas Social

Coffee

Coffee break is 11ish. Coffee for the Chicas Espresso machine is bought on an honesty basis dependent on how guilty you feel about your consumption.

Special coffee break is Thursdays when cake is provided and everyone is encouraged to make a special effort to turn up.

Friday Evening

At around 5pm on Friday, there may be a trip to Fylde Bar downstairs for a drink before the weekend. They serve alcoholic and non-alcoholic drinks as well as teas and coffees.
